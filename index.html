Trigger when scrolling by

The animate.js allows you to trigger any animation as soon as the user reaches an .animate--js element. It allows for scrolling aware animation triggering. The .js-file is lightweight and only requires jQuery.

Customize & pre-define properties

Every property of the animation is changeable; for the whole document, separate sections and areas or individually. Change it in your CSS file or via pre-defined classes.

Built with CSS custom properties

The foundation of animations.css are custom properties - an powerful addition to CSS3. This allows for high customization and readability.

Sequences

animations.css provides you with a CSS only sequence function. This means that one universal trigger will call the inherited animation to all children. The delay of each animation can be altered with the --sequence-steps variable.

Setup

How to get animations.css running

Option A: Download the files

Luckily the library is very lightweight and can be downloaded here. The package contains the animations.min.css and an animate.min.js file. While the JS file is optional (but recommended) the CSS file is mandatory.

Then link them to your project:

                
                    <!-- in the <head> -->

                    <link rel="stylesheet" href="path/to/animations.min.css">
                
            
                
                    <!-- at the end of the page -->

                    <script src="path/to/animate.min.js"></script>

Option B: Link to the files directly

If you desire to stay updated, don't want or can't implement CSS and JS files to your project you can link to them as well.

                
                    <!-- in the <head> -->

                    <link rel="stylesheet" href="...">
Use

How to use animations.css

Animate whenever

An animation normally starts when the CSS file has been loaded. It doesn't need to be visible for the user to be animated.

                
                    <!-- basic fade animation -->

                    <div class="animate"></div>

We can now add further options, for example:

                
                    <!-- looping rotateFull animation  -->

                    <div class="animate rotateFull loop"></div>

Animate when scrolled to

A more dynamic and useful approach is the .animate--js class, that let's you start an animation when the user reaches the element.

                
                    <!-- basic fade animation -->

                    <div class="animate--js"></div>

We can now add further options, for example:

                
                    <!-- delayed cutIn animation -->

                    <div class="animate--js cutIn delay-s"></div>
